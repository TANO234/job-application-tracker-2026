# Job Application Tracker — Python + Power BI

I built this because I wanted to actually understand 
what's happening when you apply for jobs. Not just 
spray and pray — but track it, analyse it, and figure 
out what's actually working.



## What I built

A Python script that generates and analyses job application 
data, calculates conversion rates at every stage of the 
recruitment funnel, and pipes everything into a Power BI 
dashboard.

First proper end-to-end data project. Learned pandas, 
matplotlib, and Power BI building this.



## What I found out

- 42% of applications just ghost you
- LinkedIn converted at 30% — best source by far
- Glassdoor converted at 0% — complete waste of time
- Only 1 in 4 interviews leads to an offer



## Tools used

- Python — pandas, matplotlib
- Power BI Service
- CSV as the data pipeline between the two



## How to run it

Install dependencies:
pip install pandas matplotlib

Run the script:
python3 job_tracker.py

Load job_applications.csv into Power BI and you're good.
